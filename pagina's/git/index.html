Hoe houd je je pagina up-to-date met git?

Commit

Commit je werk in de codespace.

Push

Push je werk door in de codespace in de terminal het commando git push uit te voeren.

Sync

Synchroniseer je werk door in de repository op github op de knop Sync te klikken.

Pull

Update je repository door in de codespace in de terminal het commando git pull uit te voeren.
